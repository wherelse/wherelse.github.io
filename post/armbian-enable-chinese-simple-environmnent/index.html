Enable the Chinese Simple environment in Armbian

  • After Phicomm N1 is installed with Armbian, whether it is a command or a browser under the desktop environment, the Chinese display is in a state of garbled characters.
    • There are two types of changing Chinese display: 1. Full Chinese display 2. Half Chinese display
      The full Chinese display is displayed in Chinese in both the desktop environment and the command environment, while the half-Chinese display can normally access Chinese content in the desktop environment, but the language of the system itself will not change.
    • For ease of use, it is recommended to enable half-Chinese display.
      The specific enabling steps are as follows:
locale
#View environment variables
sudo nano /etc/locale.gen
#Open font library, remove #save in front of zh_CN.UTF-8
sudo apt-get update
#Update program list
sudo apt-get upgrade
#update program
sudo apt-get install ttf-wqy-zenhei
#Install Chinese fonts
sudo fc-cache -v
#Update font list
sudo nano ~/.profile
#Add the corresponding content at the end of the file:
#All Chinese environment adds export LC_ALL=zh_CN.UTF-8
#Add export LC_CTYPE=zh_CN.UTF-8 and export LANG=en_US.UTF-8 to semi-Chinese environment
sudo locale-gen
#Generate locale file
sudo locale-gen zh_CN.UTF-8
# Can not be executed (Chinese can be switched on the login interface after restarting)
sudo reboot
#Restart the system

After completing the above operations, the restarted system can display Chinese normally.
