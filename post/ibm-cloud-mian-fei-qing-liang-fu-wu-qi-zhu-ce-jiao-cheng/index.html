IBM Cloud free lightweight server registration tutorial

First go to the IBM Cloud website, https://cloud.ibm.com/, and select Create Account.

Enter your email address for account verification and set a password, here it is recommended to use gmail and outlook and other foreign mainstream email addresses. After setting the password, a verification code will be sent to your mailbox for account verification. After the verification is completed, enter your personal information to complete the account registration.

After completing the registration you will enter the control panel interface of IBM CLOUD

Select the public item in Cloud Foundary as shown below to enter the Lightweight Server creation screen.

Click Create in the upper right corner.

Then go to the configuration selection screen. Here only Dallas the location is free, while you can choose the memory capacity, there are 64/128/256M 3 capacity options, each account a total of 256M light server capacity, you can choose to create a 256M machine, or multiple 64M or 128M machines, here according to their own needs to choose.

The server has multiple running resources to choose from, here you can choose according to the application you need to run. Also enter the host name, this name is unique and will be used subsequently for the host connection. After filling in the form, click Create and you're done creating the lightweight server.

Then you will be taken to the configuration guide screen of the host, which tells you how to subsequently configure and use the host.

Although this IBM Cloud's lightweight server memory is small and relatively restrictive, but can basically meet some simple use needs such as some timed tasks, crawlers and proxy services, while registration is simple, do not need to use credit card verification, or quite good. At the same time, you can choose multiple machines also have more flexibility.
