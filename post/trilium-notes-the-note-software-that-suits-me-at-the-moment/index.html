trilium notes, the note software that suits me at the moment

2022-09-07 · 5 min read
笔记

Personal feeling

I found this note/knowledge base software when I was searching for what services were running on everyone's servers, and then searched for relevant information, and it seemed to suit my appetite. Why is it suitable for me at the moment, because I am not sure whether my needs will change in the future, but it basically meets my needs at the moment. After all, there is only the most suitable, not the best.
I think there are mainly the following advantages:

  1. Natively supports various commonly used functions such as formulas, tables, mermaid diagrams, etc., and the rich text editor also supports markdown syntax.
  2. The tree-like multi-layer nested note structure is extremely convenient for notes, and supports references, links, etc. between notes. The native image management based on the database does not have a lot of troublesome problems in processing images like markdown-based software. (markdown is good anywhere, but the pictures are really a headache)
  3. Supports local use and server deployment, natively supports multi-terminal synchronization, and also supports direct use on the browser side, which makes it easy to use even when the company computer cannot install software at will, which directly adds points to me.

Install and deploy

Tilium does not officially support Chinese, but some domestic developers have provided a Chinese version: https://github.com/Nriver/trilium-translation
At the same time, a docker image is also provided, which can be deployed conveniently. For details, see: [https://hub.docker.com/r/nriver/trilium-cn](https://hub.docker.com/r/nriver/trilium -cn)

Here I use the command to deploy the docker image directly.

(Docker installation can refer to: https://yeasy.gitbook.io/docker_practice/install):

First pull the image:

docker pull nriver/trilium-cn

Run the mirror:

docker run -d --name trilium-server \
     --rm \
     -p 8080:8080 \
     -v /home/trilium-data:/root/trilium-data \
     -e TRILIUM_DATA_DIR=/root/trilium-data \
     nriver/trilium-cn

     % port, the data path can be modified as needed

In this way, it should be accessed directly through address: port. I also configured reverse proxy and https with caddy, which is convenient to use. Postscript

  1. Diary function: One-click to generate the diary notes of the day, which is very suitable as a diary application or work record.
  2. Webpage clipping: Install a chrome extension to realize one-click webpage clipping. It supports two ways of clipping pictures and webpage content directly. Recording artifact, chrome extension: [Trilium Web Clipper](https://chrome. google.com/webstore/detail/trilium-web-clipper/dfhgmnfclbebfobmblelddiejjcijbjm)
  3. Note cloning function: You can clone a note in another location for easy viewing and organization. You can also use this function to clone the official sample script and use it directly.
  4. One-click sharing. If server deployment is used, it supports one-click generation of the page link you want to share, which is convenient for sharing, and you can customize the css style of the sub-item page.
